云端同步设置

用 Supabase 免费数据库同步多设备数据。只需设置一次，iPad、手机、电脑自动同步。

设置步骤：
1. 打开 supabase.com，免费注册并新建项目（任意名称）
2. 进入项目 → 左侧菜单 SQL Editor → 粘贴以下代码并运行：

create table if not exists vocab_cards ( sync_key text not null, data jsonb not null, updated_at timestamptz default now(), primary key (sync_key) );
3. 进入 Project Settings → API，复制 Project URL 和 anon public 密钥填入下方
